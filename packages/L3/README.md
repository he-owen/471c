# L3 Language

L3 is the high level language. It contains these functions

- Let and LetRec are used for variable bindings
- Abstract and Apply for defining and applying functions.
- Immediate values are for constants.
- Primitive is used for basic arithmetic which includes operations "+", "-", and "*"
- Branch is used for conditional expressions using "<" and "=="
- Allocate, Load, and Store are used for memory management.
